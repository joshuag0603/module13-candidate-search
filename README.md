# GitHub Candidate Search

## Description

This project is a Candidate Search Application using a Github API. 

## Installation

Run the command "npm i" in the terminal at the root level to install the npm packages for the project. Add a .env file with your Github token to connect to the API. 

## Usage

The application lets you click the plus icon to add a candidate and the minus icon to skip a candidate. You can also view your saved candidates on the saved candidates page.
